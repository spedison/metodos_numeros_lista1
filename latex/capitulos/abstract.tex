
\begin{abstract}

\noindent
Este trabalho apresenta um fluxo automatizado para análise de sinais senoidais utilizando Java, Gnuplot e LaTeX.
O \underline{objetivo é ilustrar} como integrar diferentes ferramentas para produzir relatórios científicos reprodutíveis.

Esse \textbf{trecho está} "dentro de um arquivo" \textit{externo}.

Este é o abstract de teste para ver o que dá para fazer.

\vspace{2em}
\end{abstract}
